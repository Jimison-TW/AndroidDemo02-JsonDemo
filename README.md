## 開發版本
Andorid 3.1.2 </br>
SdkVersion 27 </br>
minSdkVersion 15 </br>
targetSdkVersion 27 </br>

## 學習重點
學習在Android Studio用傳統的Json解析方式，解析三種類型的Json文字
1. 物件包物件
2. 物件包陣列
3. 陣列包物件

## 如何解析
我們會需要用到 `org.json.JSONObject` 與 `org.json.JSONArray` 這兩個物件</br>
將Json物件字串或Json陣列字串丟入對應的工具中，依照想要的取值類型，使用對應的方法即可
